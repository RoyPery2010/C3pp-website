What is C3++?

C3++ is an object-oriented extension of the C3 language. It includes modern features like classes, garbage collection, method calls, inheritance, and moreâ€”all compiled using LLVM IR for performance.

Features

  • Object-oriented syntax
  • Garbage collection support
  • Virtual method tables for polymorphism
  • LLVM IR generation
  • Custom memory pools and GC tuning
  • Support for concurrency and lock-free data structures

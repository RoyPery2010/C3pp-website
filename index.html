About

C3++ is a modern, object-oriented programming language built on top of C3. It features garbage collection, LLVM IR compilation, and support for classes, functions, and more.

Features

  • Object-Oriented Syntax
  • Automatic Memory Management
  • LLVM IR Compilation
  • Simple and Clean Syntax
  • Easy to Learn

Code Sample

class Hello {
  void greet() {
    printf("Hello, world!\n");
  }
}

void main() {
  Hello h;
  h.greet();
}

Getting Started

Clone the repo from GitHub and follow the instructions to build the compiler:

git clone https://github.com/RoyPery2010/C3pp.git
cd C3pp
make

About C3++

C3++ is a modern, object-oriented extension of the C3 language. It introduces class-based programming, automatic memory management with garbage collection, and a powerful compiler backend that emits LLVM IR.

Features

  • Class-based object-oriented design
  • Automatic garbage collection
  • Custom memory pools and adaptive memory tuning
  • Multi-threaded GC support
  • LLVM IR code generation
  • Virtual method tables and polymorphism

Hello C3++

class Greeter {
  string message;

  void setMessage(string msg) {
    message = msg;
  }

  void greet() {
    print(message);
  }
}

void main() {
  Greeter g;
  g.setMessage("Hello, C3++!");
  g.greet();
}
